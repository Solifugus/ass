Open source • Go • SAS-compatible workflow

Run practical SAS-style analytics workflows with an open-source engine.

Analyst's Statistical Suite (ASS) is a clean-room implementation focused on behavioral compatibility for everyday ETL and reporting workloads. It includes a lexer → macro → parser → runtime pipeline and executes real programs end-to-end.

View on GitHub Read the README

What you get

DATA step runtime

PDV-style execution with loops, BY-groups, arrays, merge/in=, retain, keep/drop, formats, and output control.

Core PROC support

PROC PRINT, SORT, SQL, MEANS/SUMMARY, FREQ, REG (with GLM still limited), and FORMAT for practical tabular analytics.

Macro basics

Use %let, %macro/%mend, parameterized expansion, %do loops, and conditional macro logic.

Compatibility-first testing

Corpus-backed behavior checks with feature tags and per-item metadata, executed via ass test.

Quick start

go build -o ass ./cmd/ass
go test ./...

./ass run file.sas
./ass test corpus/

Note: PROC SQL uses SQLite via cgo, so a C compiler (e.g. gcc) is required.
